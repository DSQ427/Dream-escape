# Dream-escape
Eres un niño que se queda dormido en el colegio y empiezas soñar que eres un alien que puede cambiar de forma y cambiar la realidad usando habilidades con las cuales derrotas aliens y debes encontrar pedasos de una nave y avanzar en los niveles para enfrentarte a diferentes jefes para lograr escapar del colegio y despertar de tu sueño
